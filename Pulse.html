脈搏血氣診斷

背景與目的

傳統中醫透過脈診了解人體氣血狀況，但在現代難以量化與科學分析。本專案目標為建構一套數位脈診系統，透過 PPG（光體積變化描記法）感測脈搏訊號，並結合訊號分析與 Web 技術，實現脈搏的可視化、儲存與生理資訊推估。

實作內容

系統整合硬體感測、資料分析與前後端展示技術，達成即時訊號處理與脈搏狀態呈現：

  • 訊號擷取：透過 Arduino 擷取 PPG 訊號並傳輸至電腦
  • 訊號處理：使用 Python 套件進行 FFT 頻譜分析推估心率與狀態
  • 資料儲存：透過 MariaDB 保存脈搏原始資料與分析結果
  • 網頁介面：使用 ASP.NET + WebAPI 顯示數據與即時圖表

成果與應用

使用者可透過網頁即時觀測脈搏變化與分析結果，系統具有可擴充性，未來可搭配生理模型進行體質分類、疲勞監測或遠距照護應用，並具備與醫療介面整合潛力。
